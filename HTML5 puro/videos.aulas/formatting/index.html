Para inserir operadores lógicos em nossos sites temos que usar os "simbolos" "&+lt;" e "&+gt;" sem o sinal de +

Exemplo: <div>

≤ e ≥ também são possíveis: &+le e &+ge equivalem à: less equal | greater equal

Outros símbolos interessantes são
® © ™ € £ ¥ ¢ Δ ↑ ← →↓
&+reg; &+copy; &+trade; &+euro; &+pound; &+yen; &+cent; &+Delta; &+uarr;+ &+larr; &+rarr;&+darr;+ respectivamente.

A tag para usar marca texto é a MARK
Letras pequenas tem umas semântica, por exemplo, as letras pequenas de um contrato. TAG "small"

Texto deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido mas não deve ser considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado posteriormente

Tag U também serve para sublinhar porém tem forma, mas nenhuma semântica

Texto sobrescrito

Para inserir coisas do tipo 2x-1. Tag " sup "

Texto subscrito

Para inserir coisas do tipo H20<. Tag " sub "
